# Excel-Based-Sales-Analytics-Dashboard-Data-Cleaning-KPI-Tracking-Business-Insights
Store Sales Data Analysis Dashboard built in Excel using Pivot Tables, Charts, KPIs, and interactive slicers to generate business insights and track performance trends.

This project presents a complete Store Sales Data Analysis using Microsoft Excel. The objective is to analyze sales performance, identify trends, and generate meaningful business insights using dashboards and data visualization techniques.
The dashboard helps stakeholders understand revenue patterns, product performance, customer behavior, and regional sales distribution.

Project Objectives      
Analyze overall sales performance                
Identify top-performing products and categories                  
Evaluate monthly and yearly sales trends                      
Compare regional performance                             
Provide business insights for better decision-making

Key Insights Generated                          
Identified peak sales months                        
Determined high-performing product categories               
Highlighted regions contributing maximum revenue               
Analyzed sales trends for business growth opportunities
